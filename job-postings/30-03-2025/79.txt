Job Description

Iron EagleX is seeking a highly skilled Senior Machine Learning Engineer to join our dynamic team in Crystal City, VA. The ideal candidate will specialize in AI-driven automation, prompt engineering, and machine learning model optimization to deliver innovative solutions to real-world challenges. This role requires expertise in large language models (LLMs), AI agent workflows, and the development of efficient AI solutions. The position is primarily on-site, with potential for hybrid work based on project needs. Job Duties

Include (but not limited to): Develop, optimize, and implement AI workflows with a strong focus on prompt engineering and automation.

Design and fine-tune machine learning models to enhance performance and effectiveness. Research and apply best practices for large language model (LLM) utilization, including prompt orchestration and fine-tuning techniques. Develop and integrate AI agents into data workflows, leveraging state-of-the-art frameworks. Apply advanced prompt engineering techniques such as few-shot learning, chain-of-thought reasoning, self-consistency, and retrieval-augmented generation. Utilize frameworks like LangChain and DSPy to design, test, and optimize prompts. Address token limitations, cost trade-offs, and model-specific constraints across various LLMs (ChatGPT, Claude, Llama, Mistral, etc.). Implement model fine-tuning methods, including soft prompting, adapters, and supervised fine-tuning. Develop AI-driven automation workflows using agent-based architectures and frameworks such as LangGraph and OpenAI function calling. Write, test, and maintain Python code for AI model development and integration. Utilize version control systems (Git) to manage code changes and collaboration
